# Документация

**Security Journey** — бесплатная платформа с открытым исходным кодом по кибербезопасности и большим данным.

## 🛡️ Безопасность
Практические руководства по веб-, сетевой и облачной безопасности, DevSecOps, пентесту, криптографии и комплаенсу.

## 📊 Big Data
Инженерия данных, инструменты больших данных (Spark, Kafka, Hadoop), хранилища и озёра данных, потоковая обработка.
