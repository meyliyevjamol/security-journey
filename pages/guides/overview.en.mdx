# Documentation

**Security Journey** is a practical learning platform for backend developers. Topics are organized by 🟢 Beginner → 🟡 Intermediate → 🔴 Advanced.

## 🛡️ Data Security
Practical security a backend developer must watch for when handling data: input/injection, authentication and permissions, secrets, encryption, sensitive data (PII), and secure APIs.

## 📊 Working with Big Data
Handling large volumes of data correctly: database scaling, partitioning/sharding, caching, async and queues, ETL pipelines, streaming, and storage.

Pick a section from the sidebar. Code examples use **C#/.NET + PostgreSQL** (with tabs for other languages).
