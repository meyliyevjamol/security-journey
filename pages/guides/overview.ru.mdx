# Документация

**Security Journey** — практическая платформа для backend-разработчиков. Темы разбиты по уровням 🟢 Начальный → 🟡 Средний → 🔴 Продвинутый.

## 🛡️ Безопасность данных
Практическая безопасность, на которую backend-разработчик должен обращать внимание при работе с данными: ввод/инъекции, аутентификация и доступ, секреты, шифрование, чувствительные данные (PII) и безопасные API.

## 📊 Работа с большими данными
Правильная работа с большими объёмами данных: масштабирование БД, партиционирование/шардинг, кэширование, асинхронность и очереди, ETL-конвейеры, стриминг и хранение.

Выберите раздел в боковом меню. Примеры кода — на **C#/.NET + PostgreSQL** (с вкладками для других языков).
