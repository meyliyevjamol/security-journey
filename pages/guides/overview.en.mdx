# Documentation

**Security Journey** is a free, open-source learning platform for cybersecurity and big data.

## 🛡️ Security
Practical guides on web, network and cloud security, DevSecOps, penetration testing, cryptography and compliance.

## 📊 Big Data
Data engineering, big data tools (Spark, Kafka, Hadoop), data warehouse, data lake and real-time streaming.
